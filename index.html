SKOBR.

SomeKindOfBattleRoyale is an ambitious game project that I have
been working on single-handedly.

A small-scale Battle Royale game where players are greeted with
creative freedom in making a strategy to get to the top spot.
It features a balanced, but also a bittersweet
gameplay experience.

It's a project that was aimed for me to improve and learn more
about Game Development, like creating game functions through C#,
multiplayer back-end design, 3D modelling and animation,
technical art design, and conceptualizing game design and ideas.

I plan to release little to no information about the game, for now.
